Welcome to french_trans-dev!

About

This repository contains machine-corrected French translations of public-domain Greek and Latin works.

These XML files were produced by data entry company Digital Divide Data, who OCR'd, corrected and encoded the texts in accordance with the latest EpiDoc standards. While adequate, more can and should be done to improve the XML (the -dev(elopment) suffix indicates the repository is open to improvement).
